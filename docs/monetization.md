# Monetization

## Model

Use consumable essay credits through RevenueCat.

Do not start with subscriptions in MVP.

Reason:

IELTS and TOEFL users may be temporary users. They prepare for an exam, use the product intensely, and leave.

Consumable credits fit this behavior better.

## Free Credits

Every new user gets:

```text
3 free essay credits
```

## Credit Consumption

- Essay analysis: -1 credit
- Coach report: free
- Challenge participation: free

## RevenueCat Product IDs

```text
essay_pack_5
essay_pack_10
essay_pack_25
```

## Suggested Pricing

```text
5 essay credits  -> $4.99
10 essay credits -> $7.99
25 essay credits -> $14.99
```

## Paywall Triggers

Show paywall when:

- User has 0 credits and tries to analyze an essay
- User taps "Buy Credits"
- Promotional offer is shown from a campaign

## Mentor Note

Do not hide the core value too early.

Let users experience at least 2-3 analyses before forcing payment.
